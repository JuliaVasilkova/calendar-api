# calendar-api

## Сервис "Календарь"

### Краткое описание
Сервис предоставляет возможность вести календарь событий: добавлять, редактировать, удалять.

#### Ресурс системы

Ресурс системы - это событи в календаре, вида:

```
{
    "event": ["Feed my cat"],
    "rase-at": "YYYY-MM-DD HH:MM:SS",
    "remind-at": "YYYY-MM-DD HH:MM:SS",
}
```
